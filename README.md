# To-Do List App

## Description
A task management app built with HTML, CSS, and JavaScript. It allows users to add, delete, and mark tasks as completed.

## Features
- Add and remove tasks.
- Mark tasks as completed.
- Store tasks locally in the browser (using LocalStorage).
- Clean and simple user interface.

## Technologies Used
HTML, CSS and Javascript
